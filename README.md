# SYSTEM

自分専用のステータス・クエスト管理アプリ（PWA）。ステータスウィンドウ、レベル/EXP、
デイリークエスト、期限超過ペナルティ、通知機能を備えています。

## ファイル構成

```
system-app/
├── index.html   ← アプリ本体（UIとロジック）
├── manifest.json ← ホーム画面インストール用の設定・アイコン指定
├── sw.js         ← サービスワーカー（オフライン化・通知）
├── db.js         ← 状態保存用の共通コード（index.htmlとsw.jsで共有）
└── icons/        ← アプリアイコン一式
```

## なぜ「スマホだけ」に置く方法が必要か

ホーム画面に専用アイコンで追加し、アプリを閉じていても通知を受け取れるようにするには、
ブラウザの仕様上、実際に `https://` で配信されたページとして開く必要があります。
Claudeのチャット内だけで完結させることはできないため、以下のどれかで自分専用に配信してください。
どれも他人には公開されず、URLを知っている人（＝自分だけ）が使う想定で問題ありません。

## 手順（GitHub Pagesが一番簡単）

1. GitHubで新しいリポジトリを作成（Private可）
2. このフォルダの中身（index.html, manifest.json, sw.js, db.js, icons/）をそのままリポジトリ直下にアップロード
3. リポジトリの Settings → Pages → Source を "Deploy from a branch" にして、`main` ブランチ / `/ (root)` を選択
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセス可能になります
5. そのURLをiPhone/Androidでブラウザ（Safari / Chrome）で開く

> Privateリポジトリでも GitHub Pages は動作しますが、公開URL自体は知っていれば誰でも開けます。
> 完全に非公開にしたい場合は Cloudflare Pages や Netlify（どちらも無料・Basic認証設定可）も選択肢です。

## ホーム画面に追加する

**Android（Chrome）**
1. URLを開く → 右上のメニュー → 「ホーム画面に追加」または「アプリをインストール」
2. 専用アイコンでホーム画面に追加されます

**iPhone（Safari）**
1. URLをSafariで開く（Chromeでは不可、必ずSafari）
2. 共有ボタン → 「ホーム画面に追加」
3. 専用アイコンでホーム画面に追加されます

## 通知について（正直な制約）

アプリ内の「設定」タブで通知を許可すると、アプリを開いている間は期限接近・超過を通知します。
ただし、アプリを完全に閉じた状態でも毎日確実に届く通知（バックグラウンドプッシュ通知）は、
ブラウザ標準の技術だけでは以下の制約があります。

- **Android（Chrome）**: インストール済みPWAで利用頻度が高いと、定期的なバックグラウンドチェック
  （Periodic Background Sync）が動く場合がありますが、発火タイミングはブラウザ任せで保証されません。
- **iPhone（Safari）**: バックグラウンドでの定期チェックは非対応です。アプリを開いたときにだけ、
  未達成・超過分をまとめて反映します。

**確実に「毎日必ず」通知を受け取りたい場合**の現実的な追加手段として、
[ntfy.sh](https://ntfy.sh) のような無料のシンプルなプッシュ通知サービスと、
GitHub Actions の定期実行（cron）を組み合わせる方法があります。仕組み：

1. ntfy.shアプリをスマホにインストールし、自分だけの合言葉的なトピック名を購読
2. GitHubリポジトリに、毎日決まった時刻に `https://ntfy.sh/<トピック名>` へ
   HTTPリクエストを送るだけのGitHub Actionsワークフローを設置（サーバー不要・無料）
3. アプリを開いていなくても、決まった時刻に確実に通知が届く

必要であれば、この ntfy.sh 連携部分（ワークフローファイル一式）も作成できます。

## データについて

- 保存先は端末内のIndexedDB（サービスワーカーからも読み書きできるようにするため、
  localStorageではなくこちらを採用しています）
- 「設定」タブからJSONでエクスポート／インポート可能
- 機種変更時はエクスポートしたJSONを新しい端末でインポートしてください

## カスタマイズ

- クエストの報酬EXP・報酬ステータス・ペナルティEXP・ペナルティステータスは
  クエストごとに自由に設定可能
- ペナルティの厳しさ全体は「設定」タブの倍率（0.5〜2倍）で調整可能
- ステータス名（STR/VIT/INT/FOC/PER）も「設定」タブから自分の好きな呼び方に変更可能
